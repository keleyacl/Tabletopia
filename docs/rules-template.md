## 使用说明

### 文档规范

1. **语言**：规则文档使用中文编写
2. **位置**：放在游戏子项目根目录下，命名为 `rules.md`
3. **用途**：可作为 `RulesModal` 组件的内容来源，在游戏中展示规则

### 填写指南

1. 将模板中的 `{{占位符}}` 替换为实际内容
2. 根据游戏实际情况调整章节结构
3. 保持语言简洁清晰，易于理解
4. 使用列表和分段提高可读性

### 示例

参考现有游戏的规则文档：
- `lost_cities/rules.md` - 失落的城市规则
- `azul/rules.md` - 花砖物语规则
- `splendor-duel/rules.md` - 璀璨宝石·对决规则

### 渲染建议

在 `RulesModal` 组件中渲染规则文档时：

1. 使用 Markdown 解析器（如 `react-markdown`）渲染文档
2. 保持原始格式和结构
3. 支持滚动浏览长文档
4. 提供关闭按钮

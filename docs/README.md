# Tabletopia 项目文档

## 项目简介

Tabletopia 是一个在线多人对战桌游平台，包含多款经典桌游的数字化实现。项目采用 Monorepo 架构，每个游戏作为独立子项目开发，通过统一的 Portal 聚合首页提供游戏入口导航。

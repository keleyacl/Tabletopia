# Multy Portal - 聚合首页

Multy 桌游合集的聚合入口页面，提供所有游戏的导航。

## 功能

- 展示所有可用游戏的卡片入口
- 点击卡片在新标签页打开对应游戏
- 响应式布局，适配不同屏幕尺寸

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 端口配置

| 服务 | 端口 |
|------|------|
| Portal (Vite) | 4000 |

## 技术栈

- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
